# 十七、 **延保产品能不能驻场销售？**

## **思维导图**

l **核心冲突**：自店培养（长期风控）vs 第三方驻场（短期效率）——延保怎么卖？

¡ **观点 A**：自己卖。外部人员风险不可控、易虚假承诺，阻碍内部成长，且隐性成本高。

¡ **观点 B**：外包卖。专业差异大，自店人员短期难上手，按单提成能缓解人力压力，可赋能带教。

l **参悟路径**

¡ **现象层**：延保涉及保险条款等专业，自店顾问难以快速精通，容易含糊其辞或引发客诉。

¡ **文化层**：**求真**（直面专业门槛）、**尽善**（诚信销售不夸大）、**致美**（融入品牌形象）、**大爱**（对客户长期负责）、**幸福**（员工能力成长）。

¡ **内核层**：不引入驻场销售，应建立自店专业化销售能力，平衡销售业绩、成本控制与品牌长期信誉。

l **最终结论**：延保不引入驻场销售，应投入资源培养自店人员，建立自店专业化销售能力。

## **情景引入**

精诚延保服务产品由比亚迪与保险公司联合开发，推出初期服务店人员因未充分了解产品，介绍不准确，销售不规范，销售业绩不理想，在后期理赔服务中产生纠纷时有发生。

为更好地促进精诚延保服务产品的销售，是应该花资源培养和配置自店人员进行销售，还是直接请外援——邀请第三方专业人员驻店销售？

## **冲突焦点：依赖外援还是自店培养？**

**支持"反对驻场"的观点认为：**

第一，风险不可控。驻场人员的薪资完全挂钩销售额。为了促成交易，极易出现夸大保障、模糊除外责任的行为。出了事，门店和比亚迪品牌得兜底，驻场人员早就走了。

第二，形象受损。客户进店期望的是原厂专业服务。驻场人员如果缺乏品牌文化融入，言行举止像"外包推销"，会给客户留下不正规的印象。

第三，阻碍成长。长期依赖外援，店内员工会形成"反正有专人卖，我不用学"的惰性。一旦第三方撤出，门店将陷入销售真空。

**支持"引入驻场"的观点认为：**

第一，专业门槛高。延保涉及保险条款、故障概率等精算知识，普通服务顾问很难快速精通。第三方受过系统训练，能准确处理异议，业绩提升快。

第二，缓解成本压力。中小门店人力预算紧，养一个专职延保专员成本高。驻场采用按单提成模式，把固定成本变可变成本，更符合经营现状。

第三，赋能带教。好的驻场能发挥"鲶鱼效应"，通过晨会分享、复盘，向自店员工传授话术和技巧。这不仅是卖产品，更是带队伍。

**这种争论的本质，不是"卖不卖"的问题，而是"谁来卖、怎么卖、怎么控"的系统命题。**

## **第一层思考：从现象出发，我们为什么这么做？**

延保之所以难卖，是因为它不是一个简单的实体商品，而是一份复杂的金融/服务契约。

**从"真"的角度看，事实的本质是：专业能力的差异。**

普通服务顾问每天忙于接待、派工、结算，很难有整块时间去钻研枯燥的保险条款。当客户问起细节，他们往往含糊其辞，甚至为了成交也跟着瞎承诺。

第三方驻场的专业性确实能带来短期效率，但他们不属于服务店正式员工，绩效考核和薪资激励完全与销售额挂钩。为了促成交易，极易出现虚假承诺、夸大保障范围、模糊除外责任、诱导客户签约等行为。

一旦客户后续理赔遭拒，纠纷将直接指向服务店和比亚迪品牌，而第三方销售人员可能早已离开，服务店不得不承担全部善后成本与声誉损失。

**同时，驻场模式的隐性成本被低估了。** 场地、系统权限、销售佣金，加上客诉处理和品牌补偿，综合成本不一定低于自店培养。更关键的是，长期依赖外援会阻碍内部能力成长，现有员工形成惰性，一旦第三方撤出，门店将再次陷入销售真空。

**从"真"的角度看，不能因短期效率牺牲长期风控。不引入驻场销售，投入资源建立自店专业化销售能力，才是正解。**

## **第二层思考：从文化出发，我们为什么这么做？**

精诚服务文化2.0要求我们在延保销售中坚守底线，实现品牌与客户的双赢。

**求真**：直面专业门槛，但不走捷径。延保销售需要专业知识，这确实是事实，但不能因此就引入不可控的外部力量。真正的求真是承认差距后脚踏实地去学习和提升，而不是依赖外包。

**尽善**：商业向善的核心是诚信。驻场人员为促成交易容易夸大宣传，这违背了诚信原则。自店员工销售，更容易纳入统一的合规管理体系，确保介绍准确、销售规范，从源头减少理赔纠纷。

**致美**：客户期望的是无缝衔接的品牌服务体验。驻场人员游离在体系之外，着装、话术、流程难以完全融入服务店，给客户留下"外包推销"的违和感。自店人员经过系统培训后，能够自信、清晰地解释延保价值，成为品牌专业形象的自然延伸。

**大爱**：对客户负责，对品牌负责。虚假承诺带来的理赔纠纷，伤害的是客户的信任。不引入驻场，从源头控制风险，是对客户长期利益的保护。同时，投入资源培养自店员工，是对员工成长的投资，让他们掌握一项高价值技能。

**幸福**：当自店员工通过学习和实践，从"不会卖"到"专业销售"，这种能力成长带来的是职业自信和高绩效收入。客户因获得准确介绍和可靠保障而安心。这是员工与客户双向的幸福。

## **第三层思考：从精神内核出发，我们为什么这么做？**

不引入驻场销售，是一条"先难后易"的路径。

能量闭环的设计是：

投入资源培养自店人员 → 开展系统培训（产品知识、条款解读、话术演练） → 员工逐步掌握销售技能 → 独立成单率提升 → 建立自店专业化销售能力 → **业绩稳定增长，风险可控，品牌信任加固。**

如果引入驻场，短期业绩可能提升，但长期看：

驻场销售 → 员工形成依赖不学习 → 驻场撤出后销售断崖 → 重新从零开始培养 → 前期积累的信任可能因人员更替而流失。

**关键在于：能力建设不能外包。** 延保销售能力是服务店的核心竞争力之一，必须掌握在自己手里。短期启动慢、试错成本高是必然的，但一旦能力建立，就是带不走的资产。

服务店管理者需要制定清晰的培养计划：产品知识集训、老带新跟单、案例复盘、话术打磨。当员工能够自信地面对客户，准确解释条款边界，合理地促成交易时，我们的目的就达到了。

## **制度/流程优化建议**

* **合规管控红线**：所有驻场人员必须签署合规承诺书，严禁虚假承诺和夸大宣传。一旦发现违规行为，立即取消资质并进行追责。厂端可引入录音录像抽查机制。
* **赋能传帮带机制**：强制第三方对店内员工进行定期培训。规定驻场人员在晨会、夕会中的分享义务。要求驻场顾问在进行重要销售时，必须有一名自店员工旁听或协助跟单（Shadowing）。
* **能力内化时间表**：在引入驻场之初，就要设定明确的“断奶”时间表和目标（如：3个月内自店员工独立成单率达到30%，6个月达到70%）。达不到目标的，应要求驻场方调整策略或更换人员。
* **利益绑定**：将驻场服务费或提成的一部分与“自店员工成单率”挂钩。只有自店员工学会了，驻场方才能拿到全额费用，从机制上倒逼其传授真本事。
