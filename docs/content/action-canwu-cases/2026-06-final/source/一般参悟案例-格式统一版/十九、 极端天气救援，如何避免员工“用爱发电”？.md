# 十九、 **极端天气救援，如何避免员工“用爱发电”？**

## 思维导图

* **核心冲突**：单向付出（对客户） vs 双向奔赴（对员工）——极端救援怎么平衡？
  + **观点 A**：员工应奉献。为了客户满意，辛苦是应该的，这是敬业精神，理念对员工有要求。
  + **观点 B**：公司需兜底。公司若只对客户“真善美爱”而忽视员工，就是道德绑架。需对等置换。
* **参悟路径**
  + **现象层**：员工在恶劣环境和非工作时间救援，风险高、身心疲惫。若无回报，长期必生怨气。
  + **文化层**：**真**（制度对冲风险）、**善**（流程降低焦虑）、**美**（仪式重塑价值）、**爱**（兜底解决痛点）。
  + **内核层**：建立平衡机制。只有员工心里有安全感，才能向外传递幸福感。
* **最终结论**：落实强制补偿、赋予拒绝权、建立荣誉体系。用公司的大爱，换取员工对客户的大爱。

## **情景引入**

在精诚之光服务故事的评选中，为满足客户的需求，赢得客户的肯定和信赖，常常出现服务店救援人员时常得冒着风雨、雨雪、或者节假日、非上班时间（半夜三更）去救援车辆、服务客户的服务，但服务人员却很少因此获得店端直接的肯定与奖励，甚至被视为理所当然。

我们的服务理念倡导“真善美爱”，但在售后救援的场景下，**一线员工**时常得在极端天气、加班加点服务客户和车辆，**承担了“真”的职责（真实响应）和“善”的付出（解客户之急），却往往感受不到公司对他们付出的“美”与“爱”**。

## **冲突焦点**：理念要求奉献 vs 制度需要保障

在“真善美爱”的理念下，救援往往被视为服务精神的最高体现，但在具体落实中却产生了裂痕。

**强调“奉献”的一方认为：**

第一，客户利益至上。车坏在路上客户是很无助的，我们是品牌的后盾，必须随时响应。

第二，职业精神。干这行就是这样，风雨无阻是基本要求。如果都抱怨苦累，谁来服务客户？

第三，文化践行。这就是践行“真善美爱”，为了客户的信任，个人的付出是值得的。

**强调“兜底”的一方认为：**

第一，双向奔赴。理念是对所有人的。公司如果只要求员工对客户“爱”，却对员工的牺牲视而不见，这本身就是违背理念的。

第二，现实风险。极端天气救援不仅辛苦，还有安全隐患。不能只靠口号激励，必须有实质性的保障。

第三，可持续性。如果付出得不到对等的认可和高回报，员工的热情会消耗殆尽。**这两种观点的碰撞，本质是“如何理解企业文化的完整性”以及“如何建立可持续的激励闭环”的问题。**

## **第一层思考：从现象出发，我们为什么这么做？**

只有对客户单方面的“真善美爱”，不是完整的“真善美爱”。

客户在深夜、暴雨中感到无助，这时候我们的救援是雪中送炭，这确实是爱。但执行救援的员工同样处于危险和疲惫中。如果公司只把这种救援看作是员工的职责，而不去回应员工的付出，那么这种爱就是单向的，甚至是掠夺的。

**从"真"的角度看，承认恶劣环境带来的真实风险。**

我们不能因为强调奉献，就忽略了暴雨、严寒、高温对身体的伤害，也不能忽略半夜出行对家庭生活的干扰。承认这些代价，并给予补偿，才是求真。

**不能让员工“用爱发电”，而要让公司“用爱发电”给员工。** 必须通过制度化的平衡机制，将公司对客户的“爱”与对员工的“爱”在利益、尊严和安全上进行对等置换。

## **第二层思考：从文化出发，我们为什么这么做？**

精诚文化2.0要求我们在四个维度上为员工建立坚实的后盾。

**求真**：用制度对冲环境风险。这不仅仅是发点钱，而是建立“极端天气/夜间救援强制补偿机制”。配置便利员工的硬件保障（如雨衣、防滑靴、车辆安全检测），让员工有“真”的底气。

**尽善**：用流程降低突发焦虑。建立“后方守护者”机制，救援期间店端后台保持通讯，随时关注员工位置。同时，赋予员工“拒绝对无理要求”的权利——如果客户的要求超出了安全或合理范围，员工有权拒绝，公司给员工撑腰。

**致美**：用仪式重塑价值感。设立“风雨勋章”体系。让员工知道，他的每一次冲锋，公司都看在眼里，记在心上。苦劳被看见，本身就是一种美感。

**大爱**：用兜底解决后顾之忧。设立“员工关爱兜底基金”。万一员工在救援中受伤，或者因为救援导致个人生活出现重大困难，基金能提供及时的支持。

**幸福**：高保障让危险有兜底，高尊重让苦难被看见。这是员工幸福的基石。

## **第三层思考：从精神内核出发，我们为什么这么做？**

平衡的逻辑非常清晰且必须落地：

**公司对客户的爱 = 及时响应、专业救援。**

**公司对员工的爱 = 高回报、高保障、高尊重。**

这两者是互为因果的。

如果员工觉得“这一单出去，我不仅帮了客户，公司也给了我应有的回报和安全感”，他才会发自内心地认同公司的理念。

能量闭环是：

公司提供保障和激励 -> 员工消除后顾之忧 -> 员工积极、主动地提供优质救援 -> 客户获得极致体验 -> 客户的认可进一步转化为对品牌的忠诚 -> 公司效益提升 -> 更有能力反哺员工。

如果切断对员工的关怀，闭环就会断裂，最终导致员工流失、服务质量下滑，损害的是客户体验。

## **制度/流程优化建议**

* **强制补偿机制**：明确极端天气（如台风暴雨红色预警）及夜间救援的津贴标准，按次或按工时发放，绝不拖欠。
* **安全与硬件保障**：为救援车辆配备齐全的安全物资，为技师提供高品质的防风雨装备，保障出行安全。
* **拒绝权与后方守护**：明确规定员工在危及人身安全时有权拒绝救援；店端设立专人跟踪救援状态，确保“平安出、平安回”。
* **精神荣誉体系**：设立专项表彰（如“最美逆行者”、“风雨勋章”），在晨会或月度大会上给予隆重表彰。
