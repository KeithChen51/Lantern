精诚服务新版 VI 手册

以“求真、尽善、致美、大爱、幸福”为核心理念，把服务价值转化为可识别、可执行、可复制的视觉系统。

新版 VI 不再复刻旧手册的版式和口号层级。它以二十周年品牌升级为起点，把新的核心理念放到主轴位置，让标识、字体、色彩、图形和应用模板都围绕“长期信任”重新组织。

旧表达“精于勤，诚于心”保留为历史资产。新的日常理念表达预计由十字核心框架承担，因此手册首先定义这十个字的视觉样式，再定义它与主品牌、精诚服务标识共同出现时的层级。

视觉方向不追求强科技感或装饰化热闹，而是建立一种更高级、更克制、更有服务现场质感的品牌长卷：真实图像承载信任，宋体标题承载价值，深色结构承载秩序，金色与绿色只在关键判断处出现。

一手册原则

从旧版资产继承，向新版价值系统生长

这份手册不把旧版 VI 当作束缚，也不任意抛弃已有资产。正式标识保持来源清晰；理念表达、版式气质和应用系统按新版价值框架重构。

标识不重绘

主品牌与精诚服务官方标识只使用现有文件，不用文字、CSS、描摹或生成图像替代。

理念先成形

十字核心理念是新版视觉系统的价值入口，先定义字样、排列、色彩节奏和使用边界。

应用可落地

所有版式、图像、色彩和辅助图形都必须能进入客户解释、车间看板、员工支持和数字平台。

二核心理念字样

让十个字成为新版系统的视觉主轴

十字核心理念预计取代“精于勤，诚于心”的日常理念表达。它不是一句普通口号，而是一条从事实、善意、秩序、关怀到幸福的价值路径。

求真 尽善 致美 大爱 幸福

标准稿使用方正清刻本悦宋简体。五条短线不是装饰，而是把五个价值维度固定为一条可识别的路径：服务蓝代表事实，青蓝代表选择，银灰代表秩序，暖金代表关怀，结果黄代表幸福抵达。

三品牌标识与组合

主品牌优先，精诚服务承接，核心理念解释

新版组合关系不把三者等大并列。主品牌承担归属，精诚服务承担服务识别，十字理念承担价值解释。

精诚服务横版标识

延续旧版正式资产，用于服务品牌主识别。不得拉伸、描摹、改色或与十字理念合并为新 logo。

旧版理念图形

作为历史资产保留。新版日常传播不再将它与十字核心理念并列为同级主口号。

组合层级

主品牌标识最大，精诚服务次之，十字理念最轻。三者不得等大并列。

使用场景

完整组合用于封面、发布主视觉和大型物料；小尺寸场景只保留主品牌与精诚服务。

安全距离

主品牌与服务品牌之间使用垂直分隔线或足够留白；理念层与标识至少保留 0.5A。

四色彩与字体

可信、克制、清晰，而不是热闹

八禁用与交付

让规范保护高级感，而不是制造限制感

禁用规则只服务一个目标：避免标识资产被破坏，避免十字理念变成贴纸化口号，避免新版视觉系统回到旧版物料堆叠。

禁止拉伸变形

不得横向或纵向压缩、拉伸、倾斜标识。

禁止复杂背景

不得把标识直接放在高噪声、强纹理或低对比背景上。

全红背景
禁止红色过载

行动暖红只保留历史识别与少量强调，不作为新版主视觉的大面积背景。

五色贴纸化
禁止普通 icon 化

五个价值维度不能被拆成随意五色图标，也不能与旧理念 icon 并列为同级主口号。

确认理念替代

确认“求真、尽善、致美、大爱、幸福”是否正式替代“精于勤，诚于心”的日常理念表达。

确认资产版本

确认 BYD、腾势、仰望、方程豹等主品牌与精诚服务组合的正式源文件版本。

确认授权边界

复核方正、BYD Type 等字体在 Web、视频、印刷、内部平台中的使用授权。
