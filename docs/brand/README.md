# Brand Knowledge Sources

This folder keeps only brand-value text that is intentionally available to
Lighthouse knowledge building.

The service-brand VI manual, visual identity assets, and training/project
delivery documents are not part of the Lighthouse app runtime. They have been
moved out of this app repository into the adjacent Lantern workspace.

Keep files here only when they should be read by `scripts/build-knowledge.ts`
and become part of Hermit retrieval context.
